# Análise de Sentimentos e Avaliações de Consumidores

Este projeto utiliza o **Streamlit** para exibir uma análise interativa de um conjunto de dados de avaliações de produtos. A análise inclui visualizações como distribuições de notas, distribuição de sentimentos (tonalidade), e outras métricas relacionadas às avaliações de consumidores.

## Objetivo

O objetivo deste projeto é analisar o comportamento de consumidores com base nas avaliações e sentimentos expressos nas análises de produtos. Usamos uma abordagem visual e interativa para explorar as métricas relacionadas a essas avaliações, como:
- Distribuição das notas de avaliação.
- Análise da tonalidade dos comentários (sentimentos positivos e negativos).
- Medidas de dispersão e centrais dos dados.

## Tecnologias Utilizadas

- **Streamlit**: Para criar a interface interativa.
- **Pandas**: Para manipulação de dados.
- **Seaborn** e **Matplotlib**: Para visualização de dados.
- **Python**: Linguagem de programação utilizada.
